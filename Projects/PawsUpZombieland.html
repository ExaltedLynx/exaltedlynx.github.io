Paws Up Zombieland

2D Tower Defense Game

Paws Up Zombieland is a 2D tower defense game based on Arknights in which you use up to 5 unique units to defend your base against a horde of incoming zombies.

I worked on the following:
  • The base unit scripts and each units respective mechanics, stats, and ability
  • Tile placement system for units
  • Balancing each units stats, attack range and abilities
  • Unit selection functionality and UI
  • A save/load system with save slots
  • Implemented scripts for the functionality of all of the game's UI, HUD
  • Designed each of the levels in the game
